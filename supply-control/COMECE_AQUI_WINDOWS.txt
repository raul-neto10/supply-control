🆘 HELP RÁPIDO
════════════════════════════════════════════════════════════════════════════════

❌ "Python não encontrado"
   → Instale Python 3.11+ de https://www.python.org/downloads/
   → MARQUE "Add Python to PATH"
   → Reinicie Windows

❌ "Porta 5000 em uso"
   → PowerShell: netstat -ano | findstr :5000
   → PowerShell: taskkill /PID 1234 /F

❌ "Módulo não encontrado"
   → PowerShell: pip install --upgrade -r requirements.txt

❌ "Permissão negada"
   → Clique direito na pasta → Propriedades → Segurança
   → Marque "Controle Total" para seu usuário

❌ "Não consegue conectar de outro PC"
   → PowerShell: ipconfig (encontre IPv4 Address)
   → PowerShell: netsh advfirewall firewall add rule name="Supply" dir=in action=allow protocol=tcp localport=5000
   → Acesse: http://[IP_ENCONTRADO]:5000

MAIS AJUDA:
   → Leia: DEPLOYMENT_WINDOWS_GUIA_DEFINITIVO.md (seção Troubleshooting)
   → Verifique logs: C:\webapp\supply-control\logs\app.log
