🚀 ROTAS IMPLEMENTADAS

GET  /mapeamento                    → Dashboard principal
GET  /mapeamento/equipamentos       → Listagem com filtros
POST /mapeamento/equipamentos       → Aplicar filtros
GET  /mapeamento/equipamento/<id>   → Detalhes do equipamento
GET  /mapeamento/importar           → Formulário de importação
POST /mapeamento/importar           → Processar upload
GET  /mapeamento/exportar           → Download CSV
GET  /mapeamento/api/estatisticas   → API JSON de estatísticas
GET  /mapeamento/buscar             → AJAX de busca
