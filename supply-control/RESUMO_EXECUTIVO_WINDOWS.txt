🎯 3 PASSOS PARA COMEÇAR
═══════════════════════════════════════════════════════════════════════════════

1️⃣  PREPARAR AMBIENTE

    a) Instale Python 3.11+
       → https://www.python.org/downloads/
       → MARQUE "Add Python to PATH" durante instalação
       → Reinicie Windows após instalar

    b) Copie todos os arquivos para:
       → C:\webapp\supply-control\
       → Deve conter: app.py, requirements.txt, templates/, static/, etc.


2️⃣  EXECUTAR INSTALAÇÃO

    a) Abra PowerShell como Administrador
       → Win + X → Windows PowerShell (Admin)

    b) Execute na pasta do projeto:
       → cd C:\webapp\supply-control
       → .\check_requirements.bat (verifica pré-requisitos)
       → .\install_windows.bat (instala dependências)

    c) Aguarde conclusão (2-3 minutos)
       → Resultado: "Instalação concluída com sucesso!"


3️⃣  INICIAR E ACESSAR

    a) Execute na pasta do projeto:
       → .\start_server.bat

    b) Aguarde iniciar:
       → Resultado: "Running on http://127.0.0.1:5000"

    c) Abra navegador:
       → http://localhost:5000

    d) Faça login:
       → Usuário: superadmin
       → Senha: superadmin

    ✅ PRONTO! Sistema funcionalando!

⚠️  AÇÕES OBRIGATÓRIAS ANTES DE PRODUÇÃO
═══════════════════════════════════════════════════════════════════════════════

1. ⚠️  MUDAR SENHA PADRÃO
   • Login como superadmin/superadmin
   • Menu → Minha Conta → Alterar Senha
   • Escolha senha forte (maiúsculas, números, símbolos)

2. ⚠️  GERAR CHAVE SECRETA
   • PowerShell: python -c "import secrets; print(secrets.token_hex(32))"
   • Editar app.py linha ~45
   • Substituir: app.config['SECRET_KEY'] = '[sua_chave_aqui]'

3. ⚠️  DESATIVAR DEBUG
   • Editar app.py linha ~48
   • Alterar: DEBUG = False

4. ⚠️  CONFIGURAR FIREWALL (se acessar remotamente)
   • PowerShell (Admin):
     netsh advfirewall firewall add rule name="Supply" dir=in action=allow protocol=tcp localport=5000

5. ⚠️  ATIVAR HTTPS (produção)
   • Obter certificado SSL (Let's Encrypt, etc.)
   • Configurar em app.py

6. ⚠️  AGENDAR BACKUP
   • Task Scheduler: backup_database.bat diariamente
   • Manter cópias em local seguro
