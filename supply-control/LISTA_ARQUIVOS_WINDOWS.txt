🔒 SEGURANÇA - AÇÕES OBRIGATÓRIAS
───────────────────────────────────────────────────────────────────────────────

ANTES DE COLOCAR EM PRODUÇÃO:

1. ⚠️ MUDAR SENHA PADRÃO
   • Login: superadmin / superadmin
   • Menu → Minha Conta → Alterar Senha
   • Escolha senha forte (maiúsculas, números, símbolos)

2. ⚠️ GERAR SECRET KEY
   • PowerShell: python -c "import secrets; print(secrets.token_hex(32))"
   • Copie resultado
   • Edite app.py linha ~45: app.config['SECRET_KEY'] = '[copie_aqui]'

3. ⚠️ ALTERAR DEBUG
   • Edite app.py linha ~48
   • Mude: DEBUG = False

4. ⚠️ CONFIGURAR FIREWALL (se acessar de outro PC)
   PowerShell (Admin):
   netsh advfirewall firewall add rule name="Supply" dir=in action=allow protocol=tcp localport=5000

5. ⚠️ ATIVAR HTTPS (Certificado SSL)
   • Obtenha certificado (Let's Encrypt, etc)
   • Configure em app.py

6. ⚠️ FAZER BACKUP
   • Agende: backup_database.bat (diariamente)
   • Mantenha cópias em local seguro
