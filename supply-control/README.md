# Sistema simples de controle de estoque - Insumos TI

Pequeno sistema web usando Flask + SQLite para gerenciar insumos de TI (impressoras, toners, computadores, monitores, etc.). As páginas são responsivas e acessíveis por smartphone via Bootstrap.

Como usar

1. Crie e ative um ambiente virtual (recomendado):

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Instale dependências:

```bash
pip install -r requirements.txt
```

3. Inicie a aplicação:

```bash
python3 app.py
```

Abra http://localhost:5000 no navegador ou no smartphone na mesma rede.
